# Fundamentos y lógica en Java

El presente proyecto consiste en mostrar los fundamentos lógicos, de sintaxis y de semántica 
del lenguaje de programación Java.
Se utiliza la estructura estándar que genera Java Project Manager (Maven), lo que permite 
organizar el código de manera correcta y profesional.

---

## Objetivo

El fin de este proyecto es entender los componentes básicos para la programación en Java,
como sintaxis, semantica y logica.


## Estructura del Proyecto

Dentro de la carpeta src/main/java/com/example se encuentran las clases principales del proyecto, donde cada archivo aborda un concepto fundamental del lenguaje Java. Estas clases incluyen temas como variables y constantes, tipos de datos primitivos, operadores, manipulación de cadenas, entrada de datos, estructuras condicionales y de repetición, manejo de excepciones, métodos estáticos, conversión de tipos y el uso de arreglos unidimensionales y bidimensionales.

La clase Main.java funciona como punto de entrada del programa para ejecutar ejemplos o pruebas de los diferentes conceptos implementados.

La carpeta resources está destinada a almacenar archivos de configuración u otros recursos necesarios para la ejecución del proyecto. La carpeta test está reservada para las pruebas del código.

Finalmente, el archivo pom.xml contiene la configuración del proyecto Maven, donde se definen dependencias, plugins y parámetros de compilación.

### Descripción de carpetas y archivos

1. Main.java
Clase principal del proyecto que contiene el método main. Desde aquí se ejecutan los diferentes métodos de demostración de las clases del proyecto.

2. VariablesYConstantes.java
Contiene ejemplos sobre la declaración, asignación y uso de variables, así como convenciones comunes de nombres utilizadas en Java.

3. TiposDeDatosPrimitivos.java
Demuestra los diferentes tipos de datos primitivos en Java como byte, short, int, long, float, double, char y boolean.

4. Operadores.java
Muestra el uso de operadores aritméticos, de incremento y decremento para realizar operaciones básicas entre variables.

5. ConversionDeTipos.java
Explica cómo realizar conversiones de tipos de datos, incluyendo casting implícito y casting explícito.

6. EntradaDeDatos.java
Permite leer datos desde el teclado utilizando la clase Scanner, mostrando cómo capturar diferentes tipos de datos ingresados por el usuario.

7. ManipulacionCadenas.java
Presenta ejemplos de manejo de cadenas en Java, incluyendo concatenación y uso de métodos comunes como length(), toUpperCase() y trim().

8. MetodosEstaticosSimples.java
Explica la declaración y uso de métodos estáticos, así como el paso de parámetros a los métodos.

9. EstructurasCondicionales.java
Contiene ejemplos de estructuras de control condicional como if, if-else y if-else if, utilizadas para tomar decisiones dentro del programa.

10. EstructurasDeRepeticion.java
Muestra el funcionamiento de los bucles while, do-while y for para repetir bloques de código mientras se cumpla una condición.

11. ArreglosUnidimensionales.java
Presenta la creación, inicialización y recorrido de arreglos de una dimensión, así como el uso de la propiedad length.

12. ArreglosBidimensionales.java
Demuestra la declaración y recorrido de matrices utilizando arreglos bidimensionales y bucles anidados.

13. ManejoBasicoExcepciones.java
Explica el manejo básico de errores en Java mediante el uso de try-catch, incluyendo ejemplos como división por cero o errores en la entrada de datos.

## Instalación y ejecución

1. Clonar el repositorio desde GitHub:

```bash
git clone https://github.com/tu-usuario/tu-repositorio.git
```
2. Entrar a la carpeta del proyecto:

```bash
cd FUNDAMENTOS-JAVA-LOGICA/src/java/com/example/Main.java
```
3. Ejecutar desde VS Code usando la opción Run en la clase main.
